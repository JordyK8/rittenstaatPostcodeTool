Put your file in the root of this programm folder in csv format. Make sure your file is selected in the code (index.js line 5).
It should have to following fields in this order: date, km beginning, km end, km total, start location, destination.
In the function addPostcode you can use the switch statement to add or change te postcodes you want.
When the file is created you will need to import your new csv into excel en smile because this made your life a little easier :)
